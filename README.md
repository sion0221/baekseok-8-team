# [BU] 불법 주정차 킥보드 신고 웹/앱 사이트

## 📌 프로젝트 소개

<img width="942" height="902" alt="Image" src="https://github.com/user-attachments/assets/ac09fb00-b6c2-4392-b21b-c02781d6b766" />

대학 캠퍼스 내 공유 킥보드 이용이 많아지면서, 무단 방치된 킥보드가 보행 안전을 위협하는 문제가 심화되고 있습니다.  
특히 백석대학교는 넓은 부지와 경사로가 많아 킥보드 이용률이 높고, 주요 거점에 킥보드가 상습적으로 방치되어  
교통약자를 포함한 구성원의 안전을 위협하고 있습니다.

기존 신고 수단은 업체 앱 설치나 복잡한 절차가 필요해 실효성이 낮습니다.  
이에 **[BU] Kickboard**는 캠퍼스 구성원 누구나 신속하고 직관적으로 불법 주정차 킥보드를  
신고할 수 있는 전용 웹 플랫폼으로, AI(YOLOv8)를 통한 자동 감지와 GPS 기반 위치 수집으로  
복잡한 신고 절차를 최소화하고 편리하고 안전한 캠퍼스 환경 조성에 기여하고자 기획하게 되었습니다.
